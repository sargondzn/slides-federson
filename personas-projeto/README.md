# Personas — Apresentação Design Thinking

Apresentação interativa em HTML puro sobre a ferramenta **Personas** no contexto do Double Diamond.

## Como usar

Abra o arquivo `index.html` diretamente no navegador — não precisa de servidor ou instalação.

## Navegação

- **Seta direita / Espaço** → próximo slide
- **Seta esquerda** → slide anterior
- Ou use os botões na parte inferior da tela

## Estrutura do projeto

```
personas-projeto/
├── index.html       ← apresentação completa (HTML + CSS + JS em um arquivo só)
├── images/          ← coloque aqui as fotos da Carla, Felipe e outras imagens
│   └── .gitkeep
└── README.md
```

## Colaboração

Cada colega pode clonar o repositório, abrir o `index.html` no VSCode e editar diretamente.
Para ver as mudanças, basta salvar e reabrir no navegador (ou usar a extensão **Live Server** no VSCode).
